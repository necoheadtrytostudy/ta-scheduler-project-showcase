# TA Scheduler Project Showcase

A Django-based scheduling application developed as a team project for UWM's Introduction to Software Engineering course.

> The original repository is private because it contains collaborative coursework. This repository presents the project overview and my verified contributions without publishing the team's source code.

## Project Overview

TA Scheduler helps students arrange meetings with teaching assistants while supporting different workflows for students, TAs, instructors, and administrators.

## Technologies

- Python and Django
- HTML/CSS
- Django database models
- Git and GitHub
- Unit, integration, and acceptance testing

## Main Features

- User authentication and role-based pages
- Student and staff account management
- Course management
- Attendance recording
- TA office-hour management
- Student meeting reservations
- Appointment availability views
- Duplicate and conflicting reservation prevention

## Development Process

The team followed a course-based Agile workflow involving:

- Sprint planning
- User stories and PBIs
- Feature branches and pull requests
- Code review
- Product demos
- Unit, integration, and acceptance testing
- Final presentation
